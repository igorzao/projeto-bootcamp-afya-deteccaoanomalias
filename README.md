# Detecção de Anomalias em Transações com Python

## Descrição

Este projeto tem como objetivo identificar transações anômalas em uma base de dados simulada, utilizando Python para análise, tratamento e classificação das informações.

A solução foi desenvolvida para simular um cenário de monitoramento de transações, em que operações com comportamento fora do padrão podem indicar possíveis inconsistências, erros ou situações suspeitas.

## Tecnologias utilizadas

- Python
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
